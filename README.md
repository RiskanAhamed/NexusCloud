## 📌 About
NexusCloud is a modern cloud computing platform UI built with HTML, CSS, and JavaScript. It features a dark futuristic design with interactive particle animations, glassmorphism effects, and smooth scroll animations.

## ✨ Features
- 🚀 Hero section with orbit animations
- 🎨 Dark futuristic theme with glassmorphism
- ✨ Interactive particle background
- 📱 Responsive design
- 💫 AOS scroll animations
- 📊 Dashboard page
- 💰 Pricing page
- 🛠️ Services page

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (jQuery)
- Bootstrap 5
- AOS Animation Library
- Font Awesome Icons
